# Changelog

## 0.1.0

Initial public release.

- `next_dart create <name>` — scaffolds a new next-dart project with server and Flutter app directories
- `next_dart serve` — starts the development server with hot-reload on source changes
- `next_dart keygen` — generates a fresh Ed25519 key pair and prints public/private keys
- Built on `package:args` with rich `--help` output for every sub-command
- Installable as a global Dart tool: `dart pub global activate next_dart_cli`
