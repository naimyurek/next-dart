# next_dart_cli

The CLI tool for [next-dart](https://github.com/naimyurek/next-dart) — a server-driven UI framework for Flutter.

Scaffold new projects, start the dev server, and generate cryptographic keys — all from the command line.

## Install

```sh
dart pub global activate next_dart_cli
```

## Usage

```sh
# Scaffold a new project
next_dart create my_app

# Start the dev server (inside a project)
next_dart serve

# Generate a new Ed25519 key pair
next_dart keygen
```
